Postup instalace

Požadované Prostředky

Software a nástroje

  • Visual Studio 2022 Community Edition - Primární vývojové prostředí pro aplikaci.
  • Node.js - Nezbytné pro správu závislostí front-endu a spouštění některých skriptů. Dostupné ke stažení na oficiální stránce Node.js.

Knihovny a rozšíření pro Visual Studio

Následující knihovny jsou potřebné pro plnou funkčnost aplikace:

  1. ASP.NET a webový vývoj - Balíček pro vývoj webových aplikací.
  2. Vývoj aplikací Node.js - Podpora pro Node.js v Visual Studio.
  3. Vývoj desktopových aplikací pomocí .NET - Pro případné desktopové komponenty aplikace.
  4. Data storage and processing - Balíček pro práci s daty a databázemi.

Databázové nástroje

  • MS SQL Server - Databázový server pro uložení a správu dat aplikace.
  • SQL Server Management Studio - Nástroj pro správu a údržbu databáze.

Instalační dokumentace

Kompletní postup nastavení databáze a další relevantní konfigurace jsou uvedeny v dokumentu PIS - SQL Setup.

Postup lokální instalace

  1. Stáhněte si zdrojové kódy projektu.
  2. Otevřete soubor řešení projektu (soubor .sln) ve Visual Studio 2022.
  3. Před spuštěním projektu, otevřete terminál ve Visual Studio a navigujte do složky projektu serveru pomocí příkazu cd .\TransportRegister.Server\.
  4. Proveďte následující příkazy k nastavení databáze:
    1. dotnet ef database update - aplikuje migrace databáze.
    2. dotnet run seed - spustí seeding databáze.
  5. Spuštěním projektu ve Visual Studiu (kliknutím na zelenou šipku nebo klávesou F5) zpřístupníte aplikaci lokálně.

Aplikace by nyní měla být dostupná na lokální adrese https://localhost:7201/swagger pro backend a https://localhost:5173 pro frontend.

Online verze

Produkt je také dostupný na adrese transport-register.azurewebsites.net/. Zde je možné se přihlásit stejnými údaji z tabulky uživatelů.
