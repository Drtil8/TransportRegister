Postup instalace

Požadované prostředky

  • Visual Studio 2022 Community Edition
  • Knihovny (pomocí Visual Studio Installer-u):
    1. Development for ASP.NET Web
    2. Node.js application development
    3. Developing desktop applications using .NET
    4. Data storage and processing
  • Nainstalován Node.js - nodejs.org

Popis nastavení lokální databáze (MS SQL Server a Management Studio) je blíže uveden v dokumentu PIS_-_SQL_Setup.pdf

Postup lokální instalace

  1. Stáhněte si a extrahujte zdrojové kódy
  2. Otevřete projekt v Visual Studio 2022
  3. Spusťte konzoli a zadejte příkazy:
    1. cd .\TransportRegister.Server\
    2. dotnet run delete-db ; dotnet ef database update ; dotnet run seed
  4. Spusťte projekt ve Visual Studiu (zelená šipka)

Program je nyní spuštěn a dostupný na adrese https://localhost:7201 pro backend a https://localhost:5173 pro frontend.
